ORIGRAPH

Proof-Carrying AI Output, Visible to Everyone.

A deterministic demo of how AI text can be invisibly watermarked, cryptographically signed, and anchored so any verifier can check provenance.

1. Invisible Watermark

Output text includes zero-width payload metadata that does not alter visible copy but survives normal reading flows.

2. Cryptographic Signature

The issuer signs the SHA-256 digest of the delivered text. Any verifier can recover signer identity from the signature.

3. Public Anchor

Proof records can be anchored to a chain event so verification no longer depends on trusting one backend.

Anchor Published

Proof is committed to an immutable log.

User Verifies

Extension or API confirms text hash + signer + anchor.
